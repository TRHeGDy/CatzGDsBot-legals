Privacy Policy

Last Updated: June 24, 2026

1. Introduction

This Privacy Policy explains how the Discord Moderation Bot ("the Bot", "we", "our", or "us") collects, stores, and processes information while providing moderation and server management features on Discord.

2. Information We Collect

Server Information

  • Discord Server (Guild) IDs
  • Channel IDs
  • Role IDs
  • Server configuration settings

User Information

  • Discord User IDs
  • Discord usernames and display names
  • User avatars (for moderation and appeal displays)

Moderation Data

  • Warning records
  • Ban, kick, timeout, and moderation history
  • Temporary ban information
  • Appeal submissions

Message Data

The Bot may temporarily analyze message content to detect spam, scams, phishing attempts, prohibited words, excessive mentions, excessive capitalization, and cross-channel spam. Message content is generally not permanently stored unless required for moderation logs or appeals.

3. How We Use Information

  • Server moderation
  • Spam prevention
  • Scam and phishing detection
  • Rule enforcement
  • Moderation logging
  • Handling ban appeals
  • Server administration and configuration

4. Data Storage

The Bot may store server configuration files, warning histories, temporary ban records, appeal submissions, and moderation logs for as long as necessary to provide moderation functionality.

5. Data Sharing

We do not sell, rent, or trade user information. Information may only be shared with authorized server administrators and moderators, when required by law, or to comply with legal obligations.

6. Appeals and Direct Messages

Users may submit ban appeals via direct message to the Bot. Appeals are forwarded only to the moderation team of the relevant server.

7. Data Security

Reasonable technical and organizational measures are used to protect stored data. However, no method of electronic storage or transmission is completely secure.

8. User Rights

Users may request access, correction, or deletion of stored personal data where applicable by contacting the Bot owner.

9. Children's Privacy

The Bot is not intended for individuals under the minimum age required to use Discord.

10. Third-Party Services

The Bot operates on Discord and is subject to Discord's privacy practices.

11. Changes to this Privacy Policy

We reserve the right to update this Privacy Policy at any time.

12. Contact

For privacy-related inquiries, please contact the Bot owner through the support server or official contact method.
